# ルール・スキル棚卸し

以下の全ファイルを読み取り、棚卸しレポートを作成してください。

## 対象ファイル

1. グローバル CLAUDE.md: `~/.claude/CLAUDE.md`
2. グローバル Skills: `~/.claude/commands/` 配下の全 `.md` ファイル
3. プロジェクト CLAUDE.md: 現在のプロジェクトの `CLAUDE.md`（存在する場合）
4. プロジェクト Skills: `.claude/commands/` 配下の全 `.md` ファイル（存在する場合）
5. リポジトリ CLAUDE.md: リポジトリルートの `CLAUDE.md`（存在する場合）

## レポート内容

### 1. 現在有効なルール一覧
全ルールを箇条書きで列挙し、スコープ（グローバル/プロジェクト/リポジトリ）を明記。

### 2. 現在有効なSkill一覧
全Skillを箇条書きで列挙し、スコープを明記。

### 3. 問題の検出
以下を確認して報告：
- **重複**: 同じ内容が複数箇所に書かれていないか
- **矛盾**: 互いに矛盾するルールがないか
- **曖昧さ**: 解釈が分かれそうな記述がないか
- **肥大化**: 1ファイルに詰め込みすぎていないか（目安: 50行超）

### 4. 改善提案
具体的なアクション（削除/統合/移動/書き換え）を**通し番号付きの表**で提案すること（例: 1, 2, 3...）。
ユーザーが番号で指示できるようにすること。実行するかどうかはユーザーに確認すること。
